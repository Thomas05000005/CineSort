# 02 Direction

Objectif:
- fixer la direction premium retenue avant toute implementation

Format recommande:
- objectif UX
- principes visuels
- niveau de densite
- hierarchie de l'information
- contraintes produit a ne pas casser

Garde-fous:
- desktop-first explicite
- comprehension avant nouveaute visuelle
- aucun changement de comportement sans demande explicite
