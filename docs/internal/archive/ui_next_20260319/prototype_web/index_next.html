Undo

              
Règle d’usage

              
    
                
  • Prévisualiser avant tout undo réel.
    • 
                
  • L’undo ne force jamais l’écrasement d’une cible existante.

  • Les opérations irréversibles restent signalées.

Aide rapide

        
      

      

        
Étape 1 : configure ROOT + STATE_DIR local. Teste la clé TMDb si activée.

        
Étape 2 : lance l’analyse et attends "PLAN READY".

        
Étape 3 : coche "OK" seulement pour les lignes sûres. Ajuste titre/année si besoin.

        
Étape 4 : commence avec le mode simulation activé, puis applique en réel quand tout est validé.

        
Astuce : active _review pour isoler les cas à revoir.

Aide run_id / row_id

        
      

      

        
Run ID : identifie une analyse complète. Les artefacts sont dans STATE_DIR\runs\tri_films_<run_id>.

        
Row ID : identifie une ligne de film dans le plan d’analyse (utile pour Qualité et l’analyse vidéo).

        
En usage normal, vous n’avez pas besoin de saisir ces IDs: utilisez Sélectionner….
